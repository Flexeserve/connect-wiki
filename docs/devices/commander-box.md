# Commander Box

The commander enables remote maintenance and remote control of networked temperature control systems in the sense of consistent temperature and data monitoring. Up to 32 controllers can be connected to the Commander vis the ST-Bus enabling temperature and parameters adjustments directly via the touch screen.

Once the commander is connected, it can start monitoring immediately thanks to the automatic detection feature. Measured values such as temperature, uptime and alerts etc. allows our users to have a great insight which helps them with their business workflows. Software updates for the Commander and any connected controller can be carried out via the USB interface. 

## Specifications

- Wide range power supply:
    - 100-240 V AC, 50/60 Hz
- Housing size: 170 x 120 x 48 mm
- Sealing of the housing: IP 20
- Mounting type: DIN rail, wall mounting
