Dixell

What the Controller Does​

The XR30CX controls temperature in a fridge or freezer. It turns the compressor on/off and can also run defrost and light/alarm functions. (From the manual: "Model XR30CX... is a digital thermostat with off cycle defrost...")

Basic Buttons​

  • SET - View or change the temperature setpoint
  • UP (▲) - View max temperature / increase values
  • DOWN (▼) - View min temperature / decrease values
  • DEF - Start manual defrost
  • ON/OFF - Turns controller on/off (only if enabled)

(From the manual: "SET: To display target set point... (DEF) To start a manual defrost...")

Reading Temperatures​

View current set temperature​

  1. Press SET once.
  2. Display shows the setpoint.
  3. Wait 5 seconds to return to normal display.

("Push and immediately release the SET key: the display will show the Set point value")

View minimum recorded temperature​

  1. Press DOWN (▼).
  2. Display shows Lo then the minimum temperature.

View maximum recorded temperature​

  1. Press UP (▲).
  2. Display shows Hi then the maximum temperature.

Changing the Set Temperature​

  1. Hold SET for 2 seconds.
  2. Setpoint flashes.
  3. Use ▲ / ▼ to adjust.
  4. Press SET again to save.

("Push the SET key for more than 2 seconds... To change the Set value push the arrows...")

Starting a Manual Defrost​

  1. Hold DEF for 2 seconds.
  2. Defrost begins.

("Push the DEF key for more than 2 seconds and a manual defrost will start.")

LED Indicators (What the Lights Mean)​

  • Compressor LED ON -> Cooling is running
  • Compressor LED Flashing -> Safety delay
  • Defrost LED ON -> Defrost active
  • (!) LED ON -> Alarm present
  • Light LED ON -> Light output active
  • AUX LED ON -> Auxiliary relay active
  • °C/°F LED Flashing -> In programming mode

(From the LED table in the manual)

Locking / Unlocking the Buttons​

Lock the keypad​

Hold ▲ + ▼ for 3 seconds -> PoF

Unlock​

Hold ▲ + ▼ again for 3 seconds -> Pon

("Keep pressed for more than 3 s the UP + DOWN keys... 'POF' message...")

Common Alarm Messages​

  • P1 - Room probe failure
  • HA - High temperature
  • LA - Low temperature
  • dA - Door open
  • EA - External alarm
  • CA - Serious alarm

(From the alarm table in the manual)

If an alarm clears, the message disappears automatically once conditions return to normal.

Turning the Controller On/Off​

Only works if enabled by installer.

  1. Press ON/OFF -> Display shows OFF.
  2. Press again -> Controller turns back on.

("With onF = OFF... pushing the ON/OFF key, the instrument is switched off.")

When to Call a Technician​

Call for service if:

  • You see P1, P3, P4 (probe failures)
  • Temperature alarms keep returning
  • Defrost does not complete
  • Display is blank or flashing continuously
