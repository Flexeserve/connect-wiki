# Support

If you experience an issue with Connect, our support team is here to help.

## Contact Details

- **Email:** customer.support@flexeserve.com
- **Phone UK & EU:** +44 (0)1455 638300
- **Phone US:** (833) 955-8300
- **Support Hours:** 24/7

## Before You Contact Support

Please include the following details so we can resolve your issue faster:

- Site/store name
- Unit or controller type
- Brief description of the issue
- Screenshots or error messages (if available)
- Time the issue occurred

## Escalations

For urgent operational issues, please mark your request as **High Priority** and include a callback number.
